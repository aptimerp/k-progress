# 금강공업 비즈니스 규칙

## 절대 규칙 (위반 금지)

### 데이터 무결성
- **전표 마감 후 수정 금지**: 회계 전표는 마감 처리 후 절대 수정 불가
- **ERP DB 직접 수정 금지**: 바이브코더는 SAC API 경유로만 ERP 데이터 접근
- **트랜잭션 영역**: 회계·수불·전표 → MSSQL에서만 처리

### 보안
- API 키는 서버 측에서만 사용 (클라이언트 노출 금지)
- KUMKANG_API_KEY는 .env.local에만 저장 (GitHub 업로드 금지)
- 운영 DB에 직접 접속 금지 — Playground 환경 활용
