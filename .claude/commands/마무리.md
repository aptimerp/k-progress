당신은 페이즈를 마무리합니다. **GSD에 해당 명령이 있으면 먼저 호출하고, 아래 6단계로 완료하세요.**

## 진행 단계
0. **GSD 종료 명령이 있으면 호출** (예: `/gsd:complete-milestone` 또는 `/end`)
1. **STATE.md 업데이트** — 페이즈 ✅ 마킹 + 결과 요약
2. **ROADMAP.md 마킹** — 해당 페이즈 ✅ + 다음 페이즈 🔜
3. **MEMORY.md 기록** — 핵심 학습·결정사항·재사용 가능 패턴
4. **산출물 경량 검증** — 파일 존재·무결성 체크
5. **요약 출력** — 소요 시간·산출물·미해결 이슈 구조화
6. **노션 공용 페이지 업데이트** (필수) — 아래 항목 참조

## 6단계 세부 — 노션 업데이트
- `.claude/PROFILE.md`에서 **업로드 대상 페이지 ID** 읽기
- 해당 페이지에 이번 페이즈 기록 추가 — **반드시 아래 토글 표준 형식 준수**
- Notion MCP: `mcp__notion-dueul__API-patch-block-children` 사용
- MCP 없으면 사용자에게 마크다운 블록 제공 → 수동 복사
