**Goal**: 계약 이력·원가 상세·Excel 업로드·파일 보관함 구현

## Phase 5-A — SAC API 연동 준비 (선행 협의) ⏸
**기간 목표**: 2026-07 하순 ~ 08 상순  
**착수 조건**: Phase 3 완료 + SAC API 명세 수령  
**Goal**: ERP 연동 전 데이터 모델 확정 및 연동 설계  
**작업 내용**:
1. SAC 이영호 차장과 API 명세 협의 (월차결산 포함 항목 확인)
2. 건설사업부 P&L 데이터 모델 설계 (월차결산 ERP 전산화 항목 기준)
3. K-Progress P&L 화면 와이어프레임 확정
4. Supabase P&L 테이블 스키마 선설계
5. Mock 데이터로 P&L 화면 프로토타입 제작

> ⚠️ 이 Phase는 월차결산 ERP 실데이터 없이 진행 가능 — 설계·Mock 작업만
